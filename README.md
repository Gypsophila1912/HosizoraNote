# HosizoraNote

星空のように静かな夜に、思考や感情をそっと書き留めるためのモバイルノートアプリです。

React Native (Expo) で構築されており、ローカルの SQLite データベースにノートを保存するため、**インターネット接続不要**でプライベートに利用できます。

## 主な機能

- 📝 ノートの作成・閲覧・詳細表示
- 💭 思考の選択・整理画面
- 🔒 ログイン画面による利用開始フロー
- ⚙️ 設定画面
- 📦 ローカル SQLite による永続的なデータ保存（Drizzle ORM）
- 🗂️ Zustand による軽量な状態管理

## 技術スタック

| カテゴリ | 使用技術 |
|---|---|
| フレームワーク | React Native (Expo) |
| 言語 | TypeScript |
| データベース | expo-sqlite + Drizzle ORM |
| 状態管理 | Zustand |
| ナビゲーション | React Navigation (Stack / Bottom Tabs) |

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npx expo start
```

起動後、Expo Go アプリ（iOS / Android）または エミュレーターでアプリを確認できます。

## 開発規約

### コミットメッセージの命名規則

| プレフィックス | 用途 |
|---|---|
| `feat:` | 新しい機能 |
| `fix:` | バグの修正 |
| `docs:` | ドキュメントのみの変更 |
| `style:` | 空白・フォーマット・セミコロン追加など |
| `refactor:` | 仕様に影響がないコード改善（リファクタ） |
| `perf:` | パフォーマンス向上関連 |
| `test:` | テスト関連 |
| `chore:` | ビルド・補助ツール・ライブラリ関連 |

### ブランチ命名規則

| ブランチ名 | 役割 |
|---|---|
| `main` | デプロイされているブランチ（永続） |
| `develop` | 開発の大本（永続） |
| `feature/*` | 特定機能の追加・変更 |
| `release/*` | 次にリリースする機能を置く場所 |
| `hotfix/*` | main ブランチへの即時修正 |
| `topic/*` | 複数人での機能開発用 |
